HRI · Sensor Integration
Gaze-Guided Robot Control with Tobii Pro Glasses 3

Real-time gaze data pipeline from wearable eye tracker into ROS2, with operator gaze rendered live onto robot camera imagery for human-in-the-loop supervision. Tobii Pro Glasses 3 stream ingested at 240Hz via the Glasses3 API, transformed through TF2 into robot-relevant coordinate frames, and bridged to a web operator dashboard via rosbridge WebSocket.

Robot Arm · Depth Sensing
xArm 6 + RealSense Waste Sorting

Full perception-to-grasp pipeline: RealSense D435 depth camera feeds 3D point clouds through PCL into xArm 6 pick-and-place via MoveIt2. YOLOv8 instance segmentation identifies objects on a live conveyor in real time. Camera-to-robot frame calibration for accurate 3D grasp target computation; Gazebo validated before hardware deployment.

Drone · Multimodal Sensors
Drone Inspection: Sensor Fusion Stack

Multimodal inspection system fusing thermal camera, LiDAR, IMU, and environmental sensors for offshore wind turbine inspection. Kalman filter-based fusion over REST APIs provides robust localization in dynamic offshore conditions. Depth-aware defect localization maps anomalies to real-world 3D coordinates; full stack containerized in Docker with ROS2 data ingestion.

Medical · Custom Hardware
Medical Robotics Benchtop Testbed

Used a benchtop robotic actuator to evaluate medical robotics workflows in controlled physical experiments. Imaging and camera-based measurements supported repeatable validation across anatomy-inspired models.
